# Sengoku Cars
## サイト概要
新規参入の自動車メーカーの公式サイト
### サイトテーマ
自動車メーカーのコーポレートサイト。自動車のカタログ確認や見積り、来店予約ができる。
### テーマを選んだ理由
コーポレートサイトを一から自身の手で作成したかったため。
また、自身が自動車が好きで他の自動車メーカーのサイトではどのような機能があったり、ユーザエクスペリエンスが考えられているか興味があったため。
### ターゲットユーザ
自動車を所有していない若年男性向け
### 主な利用シーン
自動車のカタログ確認、見積り、来店予約、SNS機能
